# 김비서 — 업무 대시보드

서울창업허브 글로벌 진출 프로그램 관리를 위한 종합 업무 대시보드입니다.

---

## 주요 기능

- **✅ 할 일 목록** — 우선순위별 태스크 관리 (긴급/높음/보통/낮음)
- **📅 이번 주 일정** — 요일별 업무 스케줄 (월~금)
- **📈 프로젝트 진행률** — 10개 국가별 프로그램 진행 상황 시각화
- **💰 매출 요약** — 총 지원금, 모집 현황, 협약 현황 한눈에

---

## 파일 구조

| 파일명 | 역할 |
|--------|------|
| `dashboard.html` | 메인 대시보드 페이지 (4개 섹션 UI) |
| `CLAUDE.md` | 프로젝트 문서 |

---

## 사용 기술

- **HTML5** — 시맨틱 마크업
- **CSS3** — 그리드 레이아웃, 그래디언트, 플렉스박스
- **JavaScript** — 인터랙티브 체크박스 (할 일 완료 상태 관리)

---

## 특징

✨ **깔끔한 디자인**
- 흰 배경 + 카드 레이아웃
- 둥근 모서리 (border-radius)
- 부드러운 그림자 효과

📱 **반응형 구조**
- 2열 그리드 레이아웃
- 모바일 친화적 설계

🎨 **시각적 우선순위**
- 색상으로 표현된 우선순위 (빨강/주황/초록)
- 프로그레스 바로 진행률 시각화

---

## 사용 방법

1. `dashboard.html` 브라우저에서 열기
2. 할 일 항목 클릭하여 완료 표시
3. 실시간 업무 현황 확인
