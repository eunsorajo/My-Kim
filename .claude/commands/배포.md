# 🚀 배포 명령어

## 실행 내용

### 1️⃣ 변경된 파일 확인
```bash
git status
```

### 2️⃣ 커밋 메시지 자동 생성 및 커밋
```bash
git add .
git commit -m "$(date '+%Y년 %m월 %d일 %H:%M:%S 배포')"
```

### 3️⃣ GitHub에 푸시
```bash
git push
```

### 4️⃣ 배포 완료 메시지
```
✅ 배포 완료!

📍 GitHub: https://github.com/eunsorajo/My-Kim
🌐 Vercel: https://my-kim.vercel.app

💡 변경사항이 자동으로 배포되었습니다.
```

---

## 🎯 사용 방법

1. 파일 수정 완료 후
2. Claude Code에서 `/배포` 입력
3. 자동으로 커밋 및 푸시 실행
4. Vercel에서 자동 배포

---

## ⚡ 한 줄 실행

```bash
git add . && git commit -m "$(date '+%Y년 %m월 %d일 %H:%M:%S 배포')" && git push && echo "✅ 배포 완료! GitHub → Vercel 자동 배포 시작"
```
